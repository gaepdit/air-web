# Air Web Application History

## IAIP

The [Integrated Air Information Platform](https://github.com/gaepdit/iaip) was built beginning in 2004 to manage data
for the Air Protection Branch.

## Air Reports App

Printable reports were moved out of the IAIP into a 
[standalone web application](https://github.com/gaepdit-archive/airbranch-reports/) in 2022.

## Small Business Environmental Assistance Program

IAIP tools for the Small Business Environmental Assistance Program were moved to 
[a separate application](https://github.com/gaepdit-archive/sbeap/) in 2023.

## Air Web Application — Stationary Source Compliance Program

This new Air Branch web application was
launched [on 2-Feb-2026](https://github.com/gaepdit/air-web/releases/tag/v2026.2.28) to continue migrating functionality
out of the IAIP, starting with SSCP compliance and enforcement activities. The new application incorporated the Air 
Reports App as well.

## SBEAP consolidation

[On 9-Apr-2026](https://github.com/gaepdit/air-web/releases/tag/v2026.4.9), the SBEAP application code was folded into
the Air Web App codebase.
